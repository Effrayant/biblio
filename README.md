# biblio
1. Problème à résoudre

Difficulté de suivre des oeuvres culturelles consommables dans le temps sur de nombreuses plateformes différentes. 
Trop d'oublies, de désorganisation et de redondance dans les suivis.

2. Utilisateur cible

Consomateur régulier d'oeuvres culturelles consommables dans le temps qui souhaite avoir un suivi cohérent. 

3. Proposition de valeur

Permettre aux utilisateurs de suivre leurs œuvres culturelles consommables dans un seul espace clair, afin d’éviter les oublis et la dispersion.

4. Fonctionnalités principales (5–8 max)

Must have :
Consulter et trier une liste d'oeuvre.
Créer, consulter, modifier, et supprimers une oeuvre.
Ajouter une note et un commentaire a une oeuvre.
Catégorisé une oeuvre (a voir/faire, terminé, abandonné, sortie prochaine) 
Recherche dans la liste par nom.

Bonus:
Recherche par tag.

5. Hors scope (important)

Partage de liste. (le projet étant perso il n'y aura personne d'autre avec qui partagé)
Notification (pour des oeuvres qui sortent prochainement mais déjà dans la liste).
Recommandations.
